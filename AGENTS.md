# Инструкции для репозитория

- Все пользовательские тексты, документация и комментарии пишем на русском языке.
- При изменении скриптов сохраняйте понятные названия переменных и избегайте добавления временных файлов.
- В README фиксируйте новые зависимости и особенности запуска.
- Перед коммитом проверяйте, что конфигурации и примеры не содержат чувствительных данных помимо демонстрационных.
- В сообщениях о коммитах и PR используйте краткое описание на русском.
